Pfizer MiSeq Management System

Pfizer MiSeq Management System (MMS) is a web-based tools for tracking and managing MiSeq Experiments conducted within Pfizer Oncology Research Group.

In order to facility data interoperability, controled libraries were used when creating new MiSeq Experiments (sample sheets). The MMS system can be used to create, edit, export and share MiSeq Experiments. Also, it provides a customized instant search (Ajax search) option, enabling a systemic view of the current experiments under conduction.
